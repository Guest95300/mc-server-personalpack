# AMI Pack Export: Presents

AMI created these files for the selected result group:

- `ami/taxonomy/ami-export-presents.json`
  - Use this to move the group into a category or keep similar items together.
- `kubejs/server_scripts/ami/ami-export-presents.js`
  - Use this to remove the current recipes for these items before replacing them.

## What to edit first

1. Open the taxonomy JSON if the group belongs in a different category or needs a broader match.
2. Open the KubeJS script if you want to hide or replace the recipes for this group.
3. Review the item list below before shipping the files in a pack.
